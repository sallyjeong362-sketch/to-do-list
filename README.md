# 🗓 나의 투두 트래커

카테고리별 탭과 15분 단위 일정표로 하루를 계획하고 실천 여부를 체크하는 투두리스트 앱입니다.
별도 설치 없이 `index.html`을 브라우저에서 열면 바로 사용할 수 있으며, 모든 데이터는 브라우저(localStorage)에 저장됩니다.

## 주요 기능

### 📅 상단 캘린더
- 월간 캘린더에서 날짜별 달성률을 한눈에 확인 (🏆 100% / 🌟 80%+ / 🌤 50%+ / 🌱 실천 시작)
- 날짜를 클릭하면 해당 날짜의 할 일과 일정표를 보고 편집할 수 있습니다.

### 🗂 카테고리 탭 (1~5)
- **1. 수업 / 2. 운영 / 3. 수업자료 / 4. 자기계발 / 5. 기타** 탭에서 각각 할 일을 계획하고 체크합니다.
- **기타** 탭은 식사, 은행 업무 같은 일반 개인 업무용이며 장미색으로 구분됩니다.
- 할 일은 언제든 수정(✎)·삭제(✕)할 수 있습니다.

### ⏱ 6. 일정표 (15분 단위)
- 05:00~24:00을 15분 단위로 쪼갠 타임라인에 활동을 배치합니다.
- **원하는 시간 칸을 누르면 그 자리에 배치 편집기가 열립니다** — 클릭한 칸이 시작 시간이 되고, 활동 이름(할 일 자동완성)·카테고리·소요 시간(15~120분)을 정해 Enter 또는 배치를 누르면 바로 그 자리에 블록이 생깁니다. 위 양식에 입력하고 아래로 스크롤할 필요가 없어요.
- 배치된 블록마다 **세부 작업 입력란**이 항상 보여서 칸별 메모를 일정표 위에서 바로 적고 확인할 수 있습니다.
- ✕ 버튼은 **그 15분 칸 하나만 삭제**합니다 — 여러 칸짜리 활동에서 중간 칸만 지우거나 끝을 줄이는 식으로 세밀하게 조정할 수 있고, 지운 칸에는 바로 다른 활동을 넣을 수 있어요.
- 블록의 **⇕ 핸들을 잡고 아래(또는 위)로 드래그하면 엑셀처럼 같은 활동이 그 시간까지 복제**됩니다. 예: 20:00~20:15 블록을 만들고 ⇕를 20:30까지 끌면 20:30까지 채워져요. 다른 활동이 있는 칸 앞에서 자동으로 멈춥니다.
- **✓ 체크박스를 누른 채 위아래로 드래그하면 여러 칸을 한 번에 체크/해제**할 수 있어요 (포인트도 한꺼번에 적립·차감).
- 카테고리 탭에서 할 일을 완료 체크하면 **같은 이름으로 배치된 일정표 블록도 한꺼번에 완료**되고 블록 포인트까지 함께 적립됩니다(체크 해제 시 함께 해제).
- 블록은 **카테고리별 색**으로 표시됩니다(할 일과 이름이 같으면 자동 분류, 직접 선택도 가능). 상단의 분포 바에서 카테고리별 시간 배분을 한눈에 보고 하루의 균형을 잡을 수 있어요.

### 🏆 보상 시스템 (트래커)
- 할 일 완료 **+10P**, 15분 블록 실천 **+5P**
- 포인트에 따라 레벨 성장: 🌱 새싹 → 🌿 잎새 → 🪴 가지 → 🌳 나무 → 🌲 큰나무 → 🏞 숲 → 🌈 전설의 숲
- 하루 달성률 80% 이상을 이어가면 🔥 연속 달성(스트릭)이 쌓입니다.

### 🔁 매일 루틴
- 날짜 표시줄 아래 "매일 루틴" 카드에서 매일 반복할 일을 등록·수정·삭제할 수 있습니다.
- 루틴은 등록한 날부터 **매일 자동으로 나타나고**, 날짜마다 했는지/안 했는지만 체크합니다 (실천 시 +5P).
- 할 일과 달리 **다음날로 이월되지 않습니다** — 어제 못 했어도 오늘은 새로 체크해요. 날짜별 실천 여부는 달성률과 캘린더에 그대로 반영됩니다.
- 접기/펼치기로 화면을 아낄 수 있고, 기기 간 동기화(삭제 포함)도 지원됩니다.

### 📦 자동 이월
- 전날까지 완료하지 못한 할 일은 오늘 날짜로 자동으로 넘어오며 `↪ 날짜` 태그가 붙습니다.
- 이월된 항목도 자유롭게 수정·삭제할 수 있고, 중복으로 이월되지 않습니다.

### ☁ 기기 간 동기화 (로그인 불필요)
- 헤더의 ☁ 버튼에서 **동기화 코드**를 만들면, 다른 기기(모바일 등)에서 연동 링크(`?sync=코드`)를 열거나 코드를 붙여넣는 것만으로 같은 데이터를 쓸 수 있습니다.
- 저장하면 즉시 업로드되고, **두 기기를 같이 켜두면 변경사항이 몇 초 안에 서로 반영됩니다** (활동 중엔 4초마다 확인, 쉬는 중엔 30초마다, 화면이 꺼져 있으면 쉬었다가 돌아오면 바로 동기화).
- 글자를 입력하는 중에는 화면을 건드리지 않고, 입력을 마치면 밀린 변경사항을 반영합니다.
- 연결·동기화 시 두 기기의 데이터를 **덮어쓰지 않고 합칩니다**(할 일은 합집합, 겹치는 항목은 최근 기기 우선, 포인트는 더 높은 쪽 유지) — 기존에 쓰던 데이터가 있어도 안전하게 연동됩니다.
- **삭제도 기기 간에 그대로 전파됩니다.** 한 기기에서 지운 항목은 내부적으로 '삭제됨' 표식으로 기록되어 다른 기기에서도 지워지며, 다시 살아나지 않습니다(표식은 14일 후 자동 정리).
- 데이터는 익명 JSON 저장소(jsonblob.com)에 코드로만 식별되어 보관됩니다. **코드를 아는 사람은 누구나 읽고 수정할 수 있으니 비밀번호처럼 다뤄 주세요.** (30일 이상 사용하지 않으면 서버에서 삭제될 수 있습니다 — 각 기기의 로컬 데이터는 유지됩니다.)

## 사용 방법

```bash
# 저장소를 받은 뒤 브라우저에서 열기만 하면 됩니다.
open index.html
```
