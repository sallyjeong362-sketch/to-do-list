# 🗓 나의 투두 트래커

카테고리별 탭과 15분 단위 일정표로 하루를 계획하고 실천 여부를 체크하는 투두리스트 앱입니다.
별도 설치 없이 `index.html`을 브라우저에서 열면 바로 사용할 수 있으며, 모든 데이터는 브라우저(localStorage)에 저장됩니다.

## 주요 기능

### 📅 상단 캘린더
- 월간 캘린더에서 날짜별 달성률을 한눈에 확인 (🏆 100% / 🌟 80%+ / 🌤 50%+ / 🌱 실천 시작)
- 날짜를 클릭하면 해당 날짜의 할 일과 일정표를 보고 편집할 수 있습니다.

### 🗂 카테고리 탭 (1~4)
- **1. 수업 / 2. 운영 / 3. 수업자료 / 4. 자기계발** 탭에서 각각 할 일을 계획하고 체크합니다.
- 할 일은 언제든 수정(✎)·삭제(✕)할 수 있습니다.

### ⏱ 5. 일정표 (15분 단위)
- 05:00~24:00을 15분 단위로 쪼갠 타임라인에 활동을 배치합니다.
- 활동 이름·시작 시간·소요 시간(15~120분)을 정하면 15분 블록으로 나뉘어 배치되고, 블록마다 실천 여부를 체크합니다.
- 할 일 목록에 있는 항목을 자동완성으로 불러올 수 있고, 빈 칸을 클릭하면 시작 시간이 자동 입력됩니다.

### 🏆 보상 시스템 (트래커)
- 할 일 완료 **+10P**, 15분 블록 실천 **+5P**
- 포인트에 따라 레벨 성장: 🌱 새싹 → 🌿 잎새 → 🪴 가지 → 🌳 나무 → 🌲 큰나무 → 🏞 숲 → 🌈 전설의 숲
- 하루 달성률 80% 이상을 이어가면 🔥 연속 달성(스트릭)이 쌓입니다.

### 📦 자동 이월
- 전날까지 완료하지 못한 할 일은 오늘 날짜로 자동으로 넘어오며 `↪ 날짜` 태그가 붙습니다.
- 이월된 항목도 자유롭게 수정·삭제할 수 있고, 중복으로 이월되지 않습니다.

## 사용 방법

```bash
# 저장소를 받은 뒤 브라우저에서 열기만 하면 됩니다.
open index.html
```

GitHub Pages를 활성화하면 (Settings → Pages → main 브랜치) 웹 주소로도 접속할 수 있습니다.
